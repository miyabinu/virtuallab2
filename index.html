Welcome to the ESP32 Virtual Lab Project

This project demonstrates the Maximum Power Transfer Theorem using real-time data from an ESP32 microcontroller. We vary load resistance digitally through an MCP41010 potentiometer and measure voltage and current with an INA219 sensor. The results are displayed on a live dashboard, allowing us to analyze how power output changes with resistance and find the point of maximum power transfer.
